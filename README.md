## objective
# 🎬 Netflix Content Analysis Using SQL

## 📌 Overview
This project analyzes Netflix's movies and TV shows dataset using SQL to uncover trends in content type, ratings, countries, and release patterns. It demonstrates core data analysis skills — data cleaning, aggregation, and business-question-driven querying.

## 🎯 Objectives
- Analyze the distribution of content types (Movies vs TV Shows)
- Identify the most common ratings for movies and TV shows
- Explore content by release year, country, and duration
- Categorize content based on specific criteria and keywords
- Answer real-world business questions using SQL

## 🗂️ Dataset
- Source: [Netflix Movies and TV Shows dataset](https://www.kaggle.com/datasets/shivamb/netflix-shows) (Kaggle)
- Fields: show_id, type, title, director, cast, country, date_added, release_year, rating, duration, listed_in, description

## 🛠️ Tools Used
- SQL (MySQL)
- Data Cleaning & Aggregation
- Business Problem Solving

## 🔍 Key Business Questions Answered
1. Count of Movies vs TV Shows
2. Most common rating for each content type
3. Top 5 countries with the most content
4. Longest movie duration
5. Content added in the last 5 years
6. Content by a specific director
7. TV shows with more than 5 seasons
8. Content count by genre
9. Yearly content release trends by country
10. Categorizing content based on descriptive keywords

## 📊 Key Insights
- Movies make up the majority of Netflix's content library compared to TV shows.
- The United States, India, and the UK are among the top content-producing countries.
- Certain ratings dominate the platform, reflecting Netflix's primary audience targeting.
- Content additions have grown significantly in recent years.

## 🚀 How to Run
1. Clone this repository
2. Load the dataset into your SQL environment (MySQL/PostgreSQL)
3. Run the table creation script in `netflix_sql_project.sql`
4. Execute the queries to explore the insights

## 📈 Conclusion
This project showcases how SQL can be used to turn raw entertainment data into actionable business insights — from content strategy to audience targeting — skills directly applicable to a Data Analyst role.
